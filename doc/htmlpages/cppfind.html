Name

cppfind — Extended recursive search for c++ files (cpp,h)

Synopsis

cppfind [OPTIONS]... PATTERN [SEARCHING_DIR] [-h --help] [-cs --case-sensitive] [-nf --no-filenames] [-nc --no-content] [-no --name-only]

DESCRIPTION

Extended recursive search for documents, searching in filenames, contents, case in/-sensitive, with whitespaces support and optimized for git cloned repos.

OPTIONS

-h --help

Print help text to the screen.

-cs --case-sensitive

Enables case sensitive content search (by default it is insensitive).

-nf --no-filenames

Disables searching in filenames (by default enabled).

-nc --no-content

Disables searching in file content (by default enabled).

-no --name-only

Print only filenames (by default disabled).

PATTERN

String we are searching for (requires \"\" if contains spaces).

SEARCHING_DIR

Root directory for searching. By default is used current directory.
